# Project Overview
Project-Name: DB Astro Suite
Description: A professional collection of social-media focused astrophotography tools. Seamlessly transform captures into cinematic starfield animations with Starwizz or professional Instagram exposure cards with Astrogram.
Author: Dinesh Balaji Venkataraj
Base-URL: https://sidthesloth92.github.io/db-astro-suite/
Primary-Stack: Angular, AnalogJS, Tailwind CSS, Go (CLI)

# Core Mission
"From Sensor to Social" - Bridging the gap between raw astrophotography data and shareable social media content through automation and high-fidelity generation.

# Module: Starwizz
Type: Web Application / Visual Generator
Purpose: 4K Cinematic Starfield & Galaxy Animation Generator.
Technical-Details:
- Engine: Custom starfield rendering engine.
- Output: 4K Video (Browser-recorded), Sequence based effects.
- Controls: Star density, velocity vectors, rotational dynamics, zoom levels.
Usage-Context: Creating motion backgrounds for presentations, cinematic intros for YouTube, or atmospheric social media stories.

# Module: Astrogram
Type: Web Application / Graphic Generator
Purpose: Professional Exposure Card & Caption Generator for Instagram.
Technical-Details:
- Engine: Dynamic Typography & Layout engine.
- Metadata-Input: Target, Integration Time, Equipment (Scope, Mount, Camera), Filters, Bortle Scale.
- Output: PNG/JPG (Downloadable), Structured Instagram Caption (Clipboard).
Usage-Context: Post-processing workflow step to create clear "behind-the-lens" info for social media followers.

# Module: File Grouper
Type: CLI Utility
Language: Go
Purpose: Automated dataset organization for ASIAIR and other capture systems.
Command-Syntax: `file-grouper --organize-asiair ./path/to/raw`
Value-Proposition: Semantic grouping by camera, date, and object target.
